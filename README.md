# 🐰 Easter Egg Catcher 🥚

Un joc distractiv cu tematică de Paște! Prinde ouăle colorate care cad din cer cu iepurașul tău.

## Cum se joacă
- Mișcă iepurașul stânga-dreapta pentru a prinde ouăle
- Evită ouăle stricate!
- Colectează power-up-uri pentru bonusuri speciale

## Tehnologii
- HTML5 / CSS3 / JavaScript (Vanilla)
- LocalStorage pentru salvare scoruri
- Web Audio API pentru efecte sonore
